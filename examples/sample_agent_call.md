# 示例 Agent 调用过程

## User Task

基于一个原创“雨夜异常街区”世界观，生成第一章提纲。要求有悬念、规则压力和章尾钩子。

## Retrieval Plan

任务类型：生成大纲

优先召回：

- structure pattern
- conflict pattern
- emotion module
- anti pattern

## Retrieved Knowledge

```text
[资料1] WR-001 用私人目标承载奇观入口
先给主角一个具体、私人、可行动的目标，再让异常世界规则阻挡这个目标。

[资料2] CP-001 规则拦路型开篇冲突
主角带着目标进入陌生场域，场域规则要求付出代价，小线索奖励行动，同时扩大问题。

[资料3] EM-001 陌生规则压迫链
好奇 -> 不安 -> 被迫选择 -> 小发现 -> 新禁令。
```

## Agent Output Sketch

```markdown
# 第一章提纲

## 开头状态

主角为了找回某个私人目标进入异常街区，对规则半信半疑。

## 遇到阻力

街区第一条规则要求主角先偿还一笔与亲人有关的旧账，否则无法离开。

## 小解决与信息释放

主角用行动绕开第一层交易，获得线索，但线索说明亲人不是失踪，而是被街区规则扣留。

## 结尾状态与章尾牵引

主角得到下一步路径，同时获得一个禁令式钩子：到达目标地点前不能回应身后的声音。
```

## Notes

这个示例展示的是知识召回和提纲生成关系，不要求照搬 demo 文本中的专名、角色、店铺或桥段。
