# Examples

This directory contains copyright-safe demo material for local testing and portfolio demos.

- `demo_story.md`: a short original story fragment that can be uploaded into the app.
- `sample_chapter_analysis.md`: a compact example of chapter deconstruction output.
- `sample_knowledge_package.json`: an example knowledge package shaped for AI writing agents.
- `sample_canonical_cards.json`: a reviewed canonical-card shape after raw-card merge.
- `sample_agent_call.md`: an example retrieval-and-generation trace.
- `comparison_without_vs_with_rag.md`: a side-by-side comparison of no RAG, writing-guide RAG and full writing-guide/worldbuilding/memory RAG.

The demo text is intentionally short. It is meant to show the workflow, not to replace a real long-form deconstruction job.
